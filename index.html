How to Add a New Project
  1. Create a new HTML file (e.g., your-game.html)
  2. Create game logic in js/your-game.js
  3. Use Chart.js for live plotting (already included via CDN)
  4. Add a link to the navigation bar
  5. Enjoy 60fps smooth updates!

Pro tip: Use requestAnimationFrame() for smooth animations and update charts only when needed to maintain performance.
